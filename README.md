## REACT Full Responsive Animated Portfolio 

This repository contains all the code of this portfolio.
View Demo: https://matjs.netlify.app/

In this repository you will see the use of:

* REACT 
* Sanity
* SASS
* Higher Order Components
* React-redux
* Framer-motion
* EmailJS 
* Type-writter
